# 喜鹊

[English](./README.md) / 中文

deb包到玲珑包工具。

Pica，来自拉丁语的喜鹊，意思是加强精致的生态和适应性。

## 依赖

### 构建依赖

* golang-dlib-dev (>> 1.8.0+)
* golang (>= 2:1.16)

## 安装

### 从源码构建

1. 确认安装所有依赖
2. 构建:

```bash

make build

```

3. 安装:

```bash

sudo make install

```
